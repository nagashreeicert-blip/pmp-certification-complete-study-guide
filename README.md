## PMP Interview Questions (Simple & Easy)

1. **What is a Project?**  
A project is temporary work with a start, end, and a clear goal.

2. **What does a Project Manager do?**  
Plans the work, manages the team, controls the timeline, and ensures successful delivery.

3. **What is Project Scope?**  
Scope means all the work that must be done in the project.

4. **What is a Risk?**  
Something that may happen in the future and affect the project.

5. **Who are Stakeholders?**  
People or groups who are involved in the project or affected by it.

6. **What is WBS?**
